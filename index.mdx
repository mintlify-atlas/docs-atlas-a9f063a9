Tymeslot gives you booking pages, two-way calendar sync, video room creation, embeddable widgets, and automated emails — all running on your own infrastructure under the AGPLv3 licence. It's the same codebase that powers [tymeslot.app](https://tymeslot.app), so what you self-host is what paying cloud customers use in production.

## Deploy your way

<Steps>
  <Step title="Pick a deployment method">
    Run the Docker image on any VPS, install on Cloudron, deploy to Railway, or sign up for the managed cloud at [tymeslot.app](https://tymeslot.app).
  </Step>
  <Step title="Set required environment variables">
    At minimum you need `SECRET_KEY_BASE`, `PHX_HOST`, and `POSTGRES_PASSWORD`. Everything else has sensible defaults. See the [Environment Variables](/configuration/environment-variables) reference.
  </Step>
  <Step title="Configure email">
    Set `EMAIL_ADAPTER=smtp` or `EMAIL_ADAPTER=postmark` so booking confirmations, reminders, and password resets are delivered. See [Email Configuration](/configuration/email).
  </Step>
  <Step title="Connect your calendars and video">
    Link Google Calendar, Outlook, CalDAV, and a video provider from the dashboard. Follow the [Integrations](/integrations/google) guides for OAuth app setup.
  </Step>
</Steps>

<Note>
  The first account you create on a fresh install is automatically promoted to admin and gains access to `/admin`. See [Admin Management](/operations/admin) for promoting additional admins.
</Note>
